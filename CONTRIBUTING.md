### Architecture

- The main Torii application uses [tauri](https://tauri.app/) (Typescript for user interface and Rust for desktop binary) and defines the desktop client application in the directory.
- There is a backend centrally managed by the Torii Team called Torii APi and uses [PocketBase](https://pocketbase.io/) with GoLang. This is the backend of the Torii project.
